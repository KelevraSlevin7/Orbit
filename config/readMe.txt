Create a file name "configFile.txt".

All empty lines and lines starting with a "#" are ignored.

Start an object with "{" and end it with "}".

Fill in the following values:
	ObjectNumber
	mass
	radius
    positionX
    positionY
    velocityX
	velocityY

Dont forget an equal sign ("=") before the value or the line is invalid.

Enjoy :)
